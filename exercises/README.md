# Exercices guidés - Week 1

Les exercices sont intégrés dans `notebooks/06_guided_exercises.py` pour éviter la page blanche.

## Progression

1. Exploration du dataset.
2. Filtrage pandas.
3. Agrégations `groupby`.
4. Traduction Spark.
5. Requêtes SQL.
6. Mini challenge métier.

## Règle d’animation

Chaque exercice part d’un code déjà présent. Les apprenants modifient une colonne, une modalité ou un seuil, puis interprètent le résultat.
